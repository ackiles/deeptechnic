# 2026-05-22 工作记录

## 脉冲红外项目技术初评


### 背景

BP来源：第三届NICE创赛，项目名"脉冲红外连续时间视觉智能石化行业应用"，PDF 33页。核心技术：脉冲连续摄影原理（黄铁军/北大）→ 中波红外气体检测。

### 关键判断

- 脉冲连续摄影原理是真实原始创新（Engineering 2022，50+顶会论文），5/5评分确认
- 但BP中15μm像元保持20mK NETD的宣称有物理矛盾（面积1/4 → 需解释NETD守恒机制）
- 256Hz帧率对冷却型FPA（行业常态30-60Hz）是极高要求，ROIC低温77K工作有载流子冻析问题
- 2-4μm宽谱段的"800种气体覆盖"宣称存在CO2/H2O干扰问题
- BP未提及曹汛PMVIS光谱相机方案，竞争分析有重大缺口
- 团队结构性短板：核心成员无人有红外焦平面/硬件工艺量产经验
- SpikeCV GitHub仅6 stars，3年未更新，社区活跃度远低于预期
