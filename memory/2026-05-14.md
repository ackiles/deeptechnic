- 己助科技尚未完成工商注册，目前为BP项目名称/品牌代号

### 技术研判核心发现

#### 1. BP中的市场规模数据存在夸大
- BP声称"年增量约70万、存量200-400万"，实际上把三类严重程度和治疗路径完全不同的患者群体做了加法：
  - 中枢神经损伤相关呼吸障碍（颈髓损伤10-12万+脑干卒中40-60万+TBI 6-12万）→ 约60-80万/年
  - 中枢性呼吸调控不稳定（CSA/Cheyne-Stokes）
  - 呼吸肌衰退与通气依赖（ICU-VIDD/脱机困难，约ICU患者的10-30%）
- 实际纯化后适合植入式闭环膈神经刺激器的首批适应症人群估计：年增量10-15万、存量30-50万
- 核心适应症推断：首推C1-C4完全性高位颈髓损伤（膈神经完好但信号中断）

#### 2. 闭环技术路线的学术验证
- 开环膈神经/膈肌刺激对高位颈髓损伤有效——已有FDA批准产品证实（NeuRx/Synapse Biomedical, Avery Biomedical）
- 但**闭环**膈神经刺激对人体有效性的公开文献证据为零
- 相关学术方向（Case Western Reserve的DiMarco团队）仅在动物实验中探索闭环
- 己助如果坚持闭环壁垒，需要自行做FIM临床证明

#### 3. 关键技术风险清单
- 闭环反馈信号源未明确（体内/体外/混合）
- 闭环控制时延未提供
- 电极材料/规格/植入寿命未提供
- 供电方案未提供（电池vs无线供电）
- 手术方案创伤等级未量化
- 与NMPA/CDE预沟通情况未提及

### 核心发现：闭环vs开环的临床证据现状

最关键的认知更新：

1. **目前唯一直接相关的闭环膈神经刺激论文**是Renaud 2016（FPGA生物启发控制器），但只做了MATLAB仿真+FPGA台架测试，**没有动物体内实验数据**
2. **人类临床数据中，纯开环系统（NeurRx/Avery）已有充足证据**：
   - NeuRx（Synapse Biomedical）：82%脱机率（Onders 2022, n=13，早期植入）
   - Avery Biomedical：30年临床数据，>80%脱机率（严格筛选条件下）
3. 但有**冲突证据**：Lim 2020系统综述给出负面结论，暗示现有产品存在严重不足
4. 己助闭环方案理论上可以解决开环的痛点（无法自适应、膈肌疲劳、过度刺激），**但闭环在呼吸神经调控中的价值尚无任何人体数据支持**

### 闭环开环对比科普（写入以备后续参考）

**开环膈神经起搏原理：** 固定频率的节拍器——按预设速率（如12次/分）和固定电流放电。

**开环三大致命问题：**
1. **无法自适应调频调幅** — 健康人呼吸需变（咳嗽、说话、睡眠、感染），开环只会死板按预设参数
2. **膈肌疲劳** — 不会"先探查再补力"，患者自主呼吸+起搏器同时发力导致过度负荷
3. **过度刺激致神经损伤** — 固定阈值长期不变，可能累积神经损伤

**闭环方案理论上如何解决：**
```
传感器（感知膈肌EMG/胸廓阻抗/潮气量）
    ↓ 实时信号
自适应算法（判断：该不该刺激？多大强度？啥时机？）
    ↓ 优化参数
刺激器（精准放电，吸气相工作、呼气相静默）
    ↑
  患者状态变化反馈回传感器
```

| 痛点 | 闭环解决 |
|------|---------|
| 固定速率不变 | 传感器测潮气量→自动增频或降功率 |
| 不知患者想吸气还是呼气 | 实时探测呼吸起始相→精准在吸气相放电 |
| 膈肌疲劳 | 检测收缩效能→降低负载、给恢复时间 |
| 突发（咳嗽/打喷嚏） | 异常信号检测→抑制输出→平稳恢复 |

**但关键：** 以上优势全部在理论层面。Renaud 2016的FPGA仿真论文从未在活体验证过。临床上无任何闭环膈神经刺激的RCT数据。这意味着己助如果真做出闭环系统，将是全球首例——但NMPA注册路径风险极大（需自己从FIM做临床，不能引用现有文献支持疗效主张）。
