# 子报告：核心技术路线分析

## 1. Micro-OLED / Si-OLED 技术原理与路线分类

### 基本原理
Micro-OLED（又称硅基OLED、OLED-on-Silicon、OLEDoS）是在单晶硅CMOS背板上制作OLED像素阵列的微显示技术。与传统OLED在玻璃基板上制作不同，硅基板可以将像素驱动电路直接集成在像素下方，实现极高的像素密度（通常在2000-5000 PPI，甚至更高）。

### 主要技术路线

| 技术路线 | 代表厂商 | 原理概要 | 关键挑战 |
|---------|---------|---------|---------|
| **WOLED+CF（白光+彩色滤光片）** | Sony, SeeYa, eMagin(早期) | 白色发光层+RGB彩色滤光片，每个像素包含三个子像素 | **~75%光效损失**（滤光片吸收）；需要串联(tandem)结构提高亮度，导致驱动电压高达~12V |
| **RGB OLED（直接发光）** | Samsung, Applied Materials+MicroOLED | 使用光刻工艺直接形成RGB子像素，无滤光片 | 工艺难度极高，需要精确对准三个颜色层，蒸镀Mask对准精度在微米级 |
| **dPd（直接图案化沉积，eMagin独家）** | eMagin（已被Samsung收购） | 直接在像素级别沉积RGB材料，无Mask | 设备非标，量产难度大，已被收购后整合中 |
| **Si2OLED（光刻辅助全彩）** | Applied Materials + MICROOLED | 结合光刻工艺形成子像素，可精细度更高 | 仍在开发阶段，距离量产有距离 |
| **eLEAP（JDI开发）** | JDI | 无Mask沉积技术，号称更高亮度和寿命 | 同样处于早期，尚未商业化 |

### 行业主流技术分布
- **Sony**：WOLED+CF路线，长期占据>50%全球市场份额，是Apple Vision Pro的独家供应商
- **SeeYa（视涯科技）**：WOLED+CF路线，全球第二（35.2%份额），国内第一，客户包括华为、Meta
- **eMagin**：dPd路线（RGB直接发光），2023年被Samsung Display收购
- **Samsung Display**：收购eMagin后，计划2026年上半年实现光刻Micro OLED大规模量产
- **Kopin**：主要做LCD/LCoS，也有OLED-on-Silicon产品线，份额较小

## 2. 核心技术瓶颈

### 2.1 亮度与发光效率瓶颈
这是最核心的技术挑战。

**问题本质**：
- WOLED+CF方案：白光经过RGB滤光片，2/3的光被吸收，仅约25%被利用
- 为补偿损失，需要串联（tandem）结构堆叠多个发光层，驱动电压从通常的~5V上升到~12V
- 高电压带来：功耗上升、发热严重、寿命缩短
- 为达到AR眼镜户外使用所需的>5000nit亮度，需要更大电流密度，进一步加速老化

**数据**：
- Sony的ECX344A（Vision Pro用）峰值亮度约6000nits，但在典型使用场景下需要降额
- 相比之下，传统玻璃基OLED可达数百nit，微型LCD（LCoS）的亮度也更好
- 最新进展：三星与SEL在2024年发布了基于光刻工艺的1.5万尼特全彩Micro OLED

### 2.2 彩色化方案的困境

| 方案 | 优点 | 缺点 |
|------|------|------|
| WOLED+CF | 工艺相对成熟，Sony已验证量产量 | 光效损失大，色域不如RGB直接发光 |
| RGB OLED光刻 | 理论光效高，色域优秀 | 工艺精度要求极高，光刻对准困难，良率挑战 |
| 色转换（QD-OLED变种） | 可能最经济 | 蓝光OLED激发量子点，效率问题+寿命问题 |

### 2.3 像素密度与分辨率的系统瓶颈

- 当像素密度>3000PPI（像素间距<2μm）时，蒸镀Mask的变形和热膨胀成为大问题
- 对于RGB并列结构，子像素尺寸约0.5-1μm级别，对光刻机的套刻精度要求接近半导体级
- 即使采用光刻工艺，RGB三色层的对位依然极其困难

### 2.4 良率（Yield）问题

**现状**：
- 行业良率普遍在**40-60%**左右（12英寸晶圆）
- 8英寸产线良率稍好，但单片出片数少，成本更高
- 相比之下，成熟LCD/OLED面板良率通常>90%
- 低良率直接导致：单价高（视涯2025年折合单价~249元/片，即约$35/片）
- 良率爬坡时间远超预期：视涯从2016年成立到2024年才良率稳定

**根因**：
1. 硅基底的灰尘/缺陷密度控制（CMOS晶圆洁净度）
2. OLED蒸镀的均匀性（大面积薄膜均匀度控制）
3. 封装层的致密性（薄膜封装TFE对水氧的阻隔）
4. 像素电路CMOS层的良率叠加OLED层的良率 = 低复合良率

## 3. 关键设备和材料瓶颈

### 3.1 蒸镀设备
- 高端蒸镀设备（Canon Tokki、Sunic System）产能有限
- 12英寸产线专用蒸镀机交期>12个月
- 国内替代设备（合肥欣奕华等）仍在验证中

### 3.2 硅基背板
- 通常采用180nm-65nm的成熟CMOS工艺节点（过高节点成本太高，过低无法驱动高PPI）
- 中国Foundry（中芯、华虹）对显示驱动CMOS的工艺支持仍有限
- 晶圆厂产能紧张时，小批量显示产品的排产优先级低

### 3.3 彩色滤光片与微透镜
- 微透镜阵列（MLA）可显著提高光提取效率，但工艺复杂
- RGB滤光片材料（染料/颜料）对UV和热敏感

## 4. 国内主要玩家的技术进展

### 视涯科技（SeeYa）
- 成立：2016年
- 产线：12英寸硅基OLED（合肥保税区，占地5.38万㎡）
- 全球市占率：35.2%（2024年，弗若斯特沙利文）
- 2026Q1营收：1.76亿元（同比+247%）
- 客户：华为AI眼镜核心供应商、雷鸟、XREAL、小米、Meta
- 技术路线：WOLED+CF
- 产能：已超额认购，现有产能不满足客户需求
- 财务：累计亏损超9亿元，预计2026年扭亏
- IPO状态：科创板已过会，计划募资20.15亿扩产
- 预测2025-2027营收：4.69亿/22.60亿/37.34亿元

### BOE（京东方）
- 有micro-OLED研发，但投入不及视涯
- 重点可能在OLEDoS方向？

### 清越科技 / 宏禧科技
- 清越：其硅基OLED项目因技术复杂性推迟
- 宏禧：12英寸项目一期投资20亿元，预计达产年产值30亿元

### 湖畔光电 / 芯视佳
- 小批量阶段，以军工/工业应用为主

## 5. 奥视技术的路线定位（从公开资料判断）

来源：百度知道/爱企查/集微网/公司官网
- **公司全称**：广州奥视技术有限公司（另有深圳奥视微科技有限公司，做Micro-LED）
- **成立时间**：2022年
- **最新融资**：A轮
- **核心方向**：半导体光刻技术研发，**RGB直接彩色发光硅基Micro OLED**（第二代方案）
- 正在建设光刻Micro OLED产线，预计**2026年投产**
- 公司官网 o-visiontech.com 宣称："We develop second-generation RGB Micro OLED displays using organic lithography"
- 与三星光刻Micro OLED方向一致（同年规划量产）

**重要问题**：
1. "RGB直接彩色发光"与"光刻Micro OLED"意味着奥视试图跳过WOLED+CF的第一代路线，直接进入行业公认的第二代技术
2. 但全球范围内，只有三星（收购eMagin+自身研发）和日本SEL宣称在2025-2026年实现光刻Micro OLED量产
3. 初创公司直接挑战最前沿的技术路线，技术风险和资金门槛极高
4. 产线投资未披露，但从视涯20亿/宏禧20亿的一期投资看，门槛极高
