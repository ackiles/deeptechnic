# Dimension 1: RemoteID 技术与法规环境

## 关键问题回答

### Q1: 中国民航局 RemoteID 最新标准要求是什么？

中国民用航空局（CAAC）于2023年底至2024年初发布了《民用无人驾驶航空器系统安全运行管理规定》及配套的《民用无人驾驶航空器系统实名登记管理规定》等系列规范性文件，要求：

- **实名登记**：所有民用无人驾驶航空器（含轻型、小型、中型、大型）必须进行实名登记
- **远程识别（Remote ID）**：要求具备远程识别能力的无人机能够实时广播身份信息，包括无人机序列号、位置、高度、速度、控制站位置等
- **广播方式**：支持基于WiFi/蓝牙的广播模式，以及基于4G/5G的上云传输
- **实施时间线**：新生产的无人机必须内置Remote ID功能；存量无人机可加装外置模块
- **数据存储**：飞行数据需按要求保存一定期限，供监管部门查验

与美国FAA的Remote ID规则（2023年9月全面生效）类似，均要求无人机在飞行过程中广播身份与位置信息。中国CAAC的规定充分考虑了对已存量大疆等产品的兼容性——允许通过加装外置模块来满足合规要求。

### Q2: RemoteID 模块的技术方案有哪些？

主要技术方案对比：

| 方案 | 通信方式 | 优点 | 缺点 |
|------|----------|------|------|
| WiFi Broadcast | WiFi Beacon | 成本低，技术成熟 | 通信距离有限(~200m) |
| Bluetooth 5+ | BLE广播 | 功耗低，手机可接收 | 距离短(~100m) |
| 4G LTE-M/NB-IoT | 蜂窝网络 | 覆盖广，不受距离限制 | 需SIM卡，有月费成本 |
| 5G | 5G NR | 低延迟，高带宽 | 成本较高 |
| **混合方案** | WiFi+4G | 兼具近距离广播和远程上云能力 | 设计复杂度中等 |

**广土科技的方案**：集成4G上云、GPS定位、气压计、WiFi——这是一个"混合方案"设计，覆盖了近距离WiFi广播和远程4G上云两种场景，方案思路合理。GPS+气压计提供冗余定位。

### Q3: 国内外已量产 RemoteID 模块的公司有哪些？

**国际竞品**：
- **DJI（大疆）**：大疆旗下所有新型号无人机（如Mavic 3系列、Air 3、Matrice系列）均已内置Standard Remote ID
- **Aeroscope**：大疆推出的地面Remote ID接收器
- **uAvionix**：美国公司，推出Remote ID广播模块
- **Dronetag**：捷克公司，推出Remote ID模块（支持BLE + WiFi）
- **Holybro**：推出低成本Remote ID模块

**国内竞品**：
- **大疆创新**：无人机制造商，已全系预装Remote ID
- **极飞科技**：农业无人机龙头，已预装Remote ID
- **普宙飞行器**、**科比特航空**等国内无人机企业也陆续推出兼容Remote ID的产品
- 在独立的Remote ID收发模块市场中，上述国际品牌的产品已进入中国市场

### Q4: 广土科技 RemoteID 方案竞争力评估

**技术方案合理性**：中等偏上
- 集成4G+GPS+气压计+WiFi的混合方案符合行业主流思路
- "收发一体"（机载端+地面站接收解析）的设计定位较为全面

**竞争力评估**：中
- 优势：响应国内标准带来的增量市场机会；模块化方案可用于存量无人机改造
- 不足：DJI已全系预装，替代了一个主要的潜在客户群；独立Remote ID模块市场空间有限（预计2025-2026年将逐步饱和）
- 定价和成本控制是关键——这个领域的竞争者多来自珠三角低成本制造基地，芜湖地区在供应链成本方面不具备优势

## 充分性判断：基本充分
- 法规框架清晰，技术方案有明确对标
- 但中国CAAC最新修订文件的精确条文未能从网络直接获取（政府网站无法访问）
- 建议后续尽调中要求创始人提供其RemoteID产品的合规认证测试报告

## 不确定性：中等
- 中国RemoteID强制实施的具体时间表和执法力度仍有不确定性
- 外置模块市场的窗口期有限（预计2024-2026年为红利期，之后市场饱和）
