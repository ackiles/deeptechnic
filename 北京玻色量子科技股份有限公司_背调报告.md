1. **相干光量子计算路线** — 室温稳定运行，无需真空超低温环境

3. **多模式运行** — 同一机型可切换标准/精准/超频三种模式
4. **光量子计算专用光纤恒温控制** — 温度精度0.001℃以内

### 核心优势
1. **融资规模创纪录** — 10亿元B轮，国内量子计算单笔最大，投资方含国家队+头部市场化机构
2. **政府背书深厚** — 北京市科委直接扶持，中关村管委会支持，十五五规划重点方向
3. **技术实力领先** — 国内唯一可长时间稳定运行的千比特级专用量子计算机
4. **产业化领先** — 已向超算中心、运营商等交付真机，深圳有规模化制造工厂
5. **技术路线差异化** — 室温运行的相干光量子计算，运营成本低，部署方便
6. **应用场景丰富** — AI、生物医药、金融、通信、能源等多个行业已验证

### 需关注的风险点
1. **技术商业化阶段** — 量子计算整体仍处早期（NISQ时代），大规模商用化时间和路径不确定
2. **产品竞争** — 光量子计算面临着超导量子计算（本源量子/IBM）和离子阱量子计算等多路线竞争
3. **核心团队深度** — 创始人文凯的学术/产业背景公开信息有限，核心技术团队细节待进一步验证
4. **知识产权** — "数十项发明专利"的具体数量、质量和覆盖范围需通过国知局查验
5. **营收规模** — 作为未上市公司，收入规模、盈利状况不透明
6. **国际出口管制** — 量子计算技术可能受中美科技竞争影响
