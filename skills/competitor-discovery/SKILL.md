# Competitor Discovery — 独立竞品发现

## Overview

BP 中的竞品分析通常是选择性对比——只比赢的、不比输的；只提成熟的国外玩家、不提同样便宜的国内玩家。

本 skill 的原则：**永远不要只根据 BP 给出的竞品列表做竞争分析。每个行业至少能找到 3-5 家 BP 没提到的竞品。**

## When to Use

- 第一轮初评 — 对BP的竞争分析章节做独立验证
- 第三轮专家 — 结合专家信息做深度竞争格局分析

## The Process

### Step 1: 读懂BP的竞品策略

先弄清楚BP写竞品分析时的策略——通常有以下几种：
- **稻草人对比** — 选一个明显更贵/更差的产品做对标，证明自己好
- **时间错位** — 用竞品3年前的产品参数对比自己现在的参数
- **降维打击** — 故意选不同技术路线的产品证明自己更有优势
- **选择性忽略** — 完全不提某类直接竞品（通常国内竞品）

### Step 2: 主动搜索竞品

不要只查BP提到的：
1. **全球搜索** — 技术关键词 + 产品关键词的联合搜索
2. **国内优先** — 国产替代完成度高的行业，国内竞品比国外更可怕
3. **学术追踪** — 同方向的学术团队是否有在孵化的初创公司
4. **专利检索** — 核心专利的引用网络会暴露竞品

### Step 3: 定量对比表

```
| 维度 | 项目BP | 竞品A | 竞品B | 竞品C |
|------|--------|-------|-------|-------|
| 价格 | 50万 | 100万(FLIR) | 35万(国产) | 60万(方案X) |
| 帧率 | 256Hz | 30Hz | 15Hz | 60Hz |
| 分辨率 | 640x512 | 640x512 | 640x512 | 320x256 |
| 气体识别 | 是 | 否 | 否 | 是 |
| 成熟度 | 实验室样机 | 量产10年 | 量产3年 | 量产2年 |
```

### Step 4: 验证BP遗漏

- BP是否未提及任何国内竞品？→ 大概率刻意省略
- 学院派项目是否未提及同方向的商业团队？→ 常见遗漏
- 是否有一个替代技术路线（BP未提到的）可能在3年内构成威胁？

## Rationalizations

| BP话术 | 反驳 |
|--------|------|
| "我们的竞品只有FLIR"/"没有直接竞品" | "主动搜索发现以下N家国内竞品…请解释" |
| "我们的方案完全不同" | "客户买的是解决方案不是技术路线，用户视角有/无替代" |
| "竞品用的是老技术" | "老技术可能更成熟、更便宜、更容易被客户接受" |

## Red Flags

1. BP中竞品对比表的所有对比都是"我方赢"
2. BP完全不提某一家明显存在的竞品
3. BP竞品章节的信息来自3-5年前
4. BP强调技术路线差异但忽略客户实际购买决策因素

## Verification

- [ ] 独立搜索了至少 3 个BP未提的潜在竞品
- [ ] 做了定量参数对比表（至少 3 个竞品）
- [ ] 识别了BP的竞品分析策略（稻草人/时间错位/选择性忽略）
- [ ] 评估了替代技术路线的威胁
